### Default

{/* prettier-ignore */}
<Text 
  color="geist-secondary"
  size={16}
  style={{ margin: "revert" }}
>MenuButton extends the <Link href="/design/button">Button component</Link>.</Text>

```
<MenuWrapper>
  <MenuButton>Actions</MenuButton>
  <Menu width={200}>
    <MenuItem onClick={() => alert('One')}>One</MenuItem>
    <MenuItem onClick={() => alert('Two')}>Two</MenuItem>
    <MenuItem onClick={() => alert('Three')}>Three</MenuItem>
    <MenuItem onClick={() => alert('Delete')} type="error">Delete</MenuItem>
  </Menu>
</MenuWrapper>
```
