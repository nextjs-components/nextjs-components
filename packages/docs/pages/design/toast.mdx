# Toast

<Text h4 Component="h2" type="secondary" style={{ fontWeight: 400 }}>
  Display an important message globally.
</Text>

<Spacer y={2} />

### Default

```
() => {
  const toasts = useToasts()

  return (
    <Button
      onClick={() => {
        if (toasts && toasts.current) {
          toasts.current.message({
            text: 'The Netherland Dwarf jumped over the fence.'
          })
        }
      }}
    >
      Show Toast
    </Button>
  )
}
```

<Spacer y={2} />

### Multiline

```
() => {
  const toasts = useToasts()

  return (
    <Button
      onClick={() => {
        if (toasts && toasts.current) {
          toasts.current.message({
            text: 'The Netherland Dwarf jumped over the fence. The Netherland Dwarf jumped over the fence. The Netherland Dwarf jumped over the fence. The Netherland Dwarf jumped over the fence.'
          })
        }
      }}
    >
      Show Toast
    </Button>
  )
}
```

<Spacer y={2} />

### Preserve

Preserved toasts will not automatically disappear.

```
() => {
  const toasts = useToasts()

  return (
    <Button
      onClick={() => {
        if (toasts && toasts.current) {
          toasts.current.message({
            text: 'The Netherland Dwarf jumped over the fence.',
            preserve: true
          })
        }
      }}
    >
      Show Toast
    </Button>
  )
}
```

<Spacer y={2} />

### Action

```
() => {
  const toasts = useToasts()

  return (
    <Button
      onClick={() => {
        if (toasts && toasts.current) {
          toasts.current.message({
            text: 'The Netherland Dwarf jumped over the fence. The Netherland Dwarf jumped over the fence again.',
            action: 'Undo'
          })
        }
      }}
    >
      Show Toast
    </Button>
  )
}
```

### Action + Cancel
